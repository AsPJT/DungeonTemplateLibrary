## Quick Start (CMake) â©

First clone this code.

```
git clone https://github.com/Kasugaccho/DungeonTemplateLibrary.git
cd DungeonTemplateLibrary/
```

### For macOS or Linux

```
mkdir build
cd build
cmake ..
make ci
```

### For Windows (VS2017)

```
mkdir build
cd build
cmake .. -G"Visual Studio 15 2017 Win64"
MSBuild DTL.sln /p:Configuration=Release
```
